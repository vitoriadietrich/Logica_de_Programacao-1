# Lógica de Programação — Algoritmos e Estruturas

Repositório destinado ao registro das atividades de Lógica de Programação realizadas no curso Técnico em Informática para Internet no Senac Lapa Tito.

## Tecnologias e Ferramentas
- **Portugol** (VisuAlg / Portugol Studio)
- **Lógica Algorítmica**

## Tópicos Estudados
- [x] Algoritmos sequenciais (entrada, processamento e saída)
- [x] Desvios condicionais (`se/senão`, `escolha/caso`)
- [x] Laços de repetição (`enquanto`, `para`, `repita`)
- [x] Variáveis, tipos de dados e operadores lógicos/aritméticos
- [x] Vetores e Matrizes (Estruturas de dados básicas)

## Objetivo
Desenvolver o raciocínio lógico e a base algorítmica necessária para transição para linguagens de programação de alto nível (como Java, Python ou Node.js), com foco total em arquitetura Back-end.
